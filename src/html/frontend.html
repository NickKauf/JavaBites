Package managers and how we all depend on one another

July 23, 2021

We learned that you can implement Bootstrap in an html page with a simple link in the header. It turns out web apps implement lots of code from other places using package managers like NPM, which says it’s the world’s largest software registry. Recently, I read about case in 2016 where one programmer withdrew a few lines of his code from this registry and it broke a ton of websites. This just shows how interdependent we’ve become.
